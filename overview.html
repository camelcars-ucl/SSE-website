What does software systems engineering involve?

Software engineering is the process of developing computer based software to benefit both industry and potentially society as a whole.

The goal of software systems engineering is to develop software that is:

  • Meets a target – ensuring that software developed meets the set specification. This is essential in a commercial environment. This is also essential in areas such as avionic software engineering (software used in aeroplanes) where software produced is used to prevent harm coming to civilians.
  • Reliable – the ability to ensure that developed software has been thoroughly tested before deployment. This is to ensure both safety when used in practical scenarios and for software to essentially work as desired. This is to make sure users are not inconvenienced and potentially endangered.
  • Adaptable – The ability to improve software to meet the changing requirements of users/companies. It also provides the ability to innovate new features for pieces of software.
  • Safe - developed software is fundamentally safe when used in real life applications
  • Economical - ensuring that you are in full control of the costs when creating software. Ensuring this is managed in a commercial environment is critical to business.
